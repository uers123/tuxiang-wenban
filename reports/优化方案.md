## 阶段一：修复 OCR 管线（评分 10% → 45%）

### 问题 1.1：图像预处理破坏图表细节

**现状**：`_preprocess_image()` 用固定阈值 180 二值化 + 中值滤波，红色散点、蓝色三角、浅色网格线全部消失。

**改造方案**：预处理管线改为**双通道**——一份做 OCR（灰度增强），一份保留颜色（做图表分析）。

```python
def preprocess_image(source: Path) -> tuple[Path, Image.Image]:
    """
    返回 (ocr_ready_path, color_image)
    - ocr_ready: 灰度增强图，供 Tesseract 识别文字
    - color_image: 保留原始颜色，供图表分析
    """
    from PIL import Image, ImageFilter, ImageOps
    img = Image.open(source)
    img = ImageOps.exif_transpose(img)

    # 颜色副本 → 给图表分析
    color_copy = img.copy()

    # OCR 副本：灰度 + 自适应增强（不硬二值化）
    ocr_img = img.convert("L")
    ocr_img = ImageOps.autocontrast(ocr_img, cutoff=2)
    ocr_img = ocr_img.filter(ImageFilter.MedianFilter(size=3))
    # 不再做硬阈值二值化，Tesseract 对灰度图效果更好

    temp = tempfile.NamedTemporaryFile(prefix="doc_textify_", suffix=".png", delete=False)
    temp.close()
    ocr_img.save(temp.name)
    return Path(temp.name), color_copy
```

**关键改变**：
- ❌ 删除 `image.point(lambda px: 255 if px > 180 else 0, mode="1")`（把图片变成纯黑白）
- ✅ 保留为灰度图，Tesseract 在灰度图上识别率不差，但信息更丰富
- ✅ 返回 color_copy，供后续图表分析

### 问题 1.2：Tesseract 页面分割模式选错

**现状**：`--psm 6`（单一均匀文本块），跨栏乱读、图表区域无法识别。

**改造方案**：自动选择 PSM 模式：

```python
def _select_psm(image: Image.Image) -> int:
    """根据图像特征自动选择 PSM 模式"""
    w, h = image.size
    aspect = w / h

    # 宽图 → 可能是多栏 → PSM 4（单列可变大小文本）
    if aspect < 0.6 or aspect > 1.4:
        return 4
    return 3  # 默认自动
```

**关键改变**：`--psm 6` → `--psm 3`（自动）或 `--psm 4`（单列可变大小）

def _ocr_with_pytesseract(image_path, lang, psm):
    """使用 pytesseract 替代裸 subprocess"""
    config = f"--psm {psm}"

### 问题 1.4：没有 PDF 渲染器（扫描件无法处理）

**现状**：`force_ocr=True` 的输出只是 placeholder。

**改造方案**：用 `pypdfium2` 把 PDF 每页渲染为图片，然后走 OCR 管线。

```python
def _render_pdf_page_to_image(pdf_path: Path, page_number: int) -> Path:
    """把 PDF 第 N 页渲染为 PNG 图片"""
    import pypdfium2 as pdfium
    pdf = pdfium.PdfDocument(str(pdf_path))
    page = pdf.get_page(page_number - 1)
    bitmap = page.render(scale=2)  # 2x 提高 OCR 效果
    pil_image = bitmap.to_pil()
    temp = tempfile.NamedTemporaryFile(prefix="doc_textify_pdf_", suffix=".png", delete=False)
    temp.close()
    pil_image.save(temp.name)
    return Path(temp.name)
```

## 阶段二：版面分析与结构重建（评分 45% → 65%）

### 问题 2.1：没有多栏检测

**现状**：所有图片只用 `--psm 6`，Tesseract 一行行吐文字，按 (y, x) 排序后直接输出，跨栏错乱。

**改造方案**：增加**投影分析**检测列边界，列内按阅读顺序排序。

```python
def detect_columns(blocks: list[Block], page_width: float) -> list[list[Block]]:
    """
    根据文字块 x 坐标分布检测多栏布局。
    将 blocks 按列分组，每列内部按 (y, x) 排序。
    """
    if not blocks or not page_width:
        return [blocks]

    # 收集所有 x 中心点
    centers = []
    for b in blocks:
        if b.bbox:
            cx = (b.bbox[0] + b.bbox[2]) / 2
            centers.append(cx)

    if not centers:
        return [blocks]

    # 用间隙检测列边界：x 方向直方图，找空白间隙
    hist = [0] * int(page_width)
    for b in blocks:
        if b.bbox:
            x0, x1 = int(b.bbox[0]), min(int(b.bbox[2]), int(page_width) - 1)
            for x in range(x0, x1):
                hist[x] += 1

    # 找连续空白区域（超过 page_width 的 8% 为列间隙）
    min_gap = page_width * 0.08
    gaps = []
    in_gap = False
    gap_start = 0
    for x in range(int(page_width)):
        if hist[x] == 0:
            if not in_gap:
                gap_start = x
                in_gap = True
        else:
            if in_gap:
                if x - gap_start >= min_gap:
                    gaps.append((gap_start, x))
                in_gap = False

    if not gaps:
        return [blocks]  # 单列

    # 按列分组
    columns = [[] for _ in range(len(gaps) + 1)]
    for b in blocks:
        if b.bbox:
            cx = (b.bbox[0] + b.bbox[2]) / 2
            col_idx = sum(1 for gs, ge in gaps if cx > ge)
            col_idx = min(col_idx, len(columns) - 1)
            columns[col_idx].append(b)
        else:
            columns[0].append(b)

    # 每列内按 (y, x) 排序
    for col in columns:
        col.sort(key=lambda b: (b.bbox[1] if b.bbox else 0, b.bbox[0] if b.bbox else 0))

    return columns
```

### 问题 2.2：标题分类不准确

**现状**：只通过文本长度和标点判断标题，不考虑字号和位置。

**改造方案**：利用 Tesseract 返回的字号信息 + 位置特征来判断。

```python
def classify_block(text: str, font_size: float | None,
                   x: float, y: float, page_width: float, page_height: float) -> str:
    """
    基于字号大小和位置进行结构化分类。
    """
    # 页眉/页脚：靠近页面顶端或底端
    if y < page_height * 0.05 or y > page_height * 0.95:
        if len(text) < 60:
            return "header" if y < page_height * 0.05 else "footer"

    # 标题：字号明显大于正文（> 14pt）或居中
    if font_size:
        if font_size >= 18:
            return "title"
        if font_size >= 14:
            return "heading"

    # 居中短文本 → 可能是标题
    center_ratio = abs(x + (font_size or 12) * len(text) * 0.5 - page_width / 2) / page_width
    if center_ratio < 0.15 and len(text) < 50 and not text.endswith((".", "。")):
        return "heading"

    return "paragraph"
```

### 问题 3.3：颜色元素提取（数据点和分类区间）

```python
def extract_colored_elements(panel_region: np.ndarray, color: tuple) -> list[dict]:
    """
    提取指定颜色的元素（散点、竖线等），结合 OCR 文字恢复类别和数值。
    """
    import cv2
    import numpy as np

    # 定义颜色范围（红色散点、蓝色三角、红色竖线）
    color_ranges = {
        "red_dot": (np.array([0, 50, 50]), np.array([10, 255, 255])),
        "blue_triangle": (np.array([100, 50, 50]), np.array([130, 255, 255])),
    }

    hsv = cv2.cvtColor(panel_region, cv2.COLOR_RGB2HSV)
    mask = cv2.inRange(hsv, *color_ranges[color])

    # 轮廓检测 → 每个元素的位置
    contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    elements = []
    for cnt in contours:
        x, y, w, h = cv2.boundingRect(cnt)
        # 将像素坐标映射回数据坐标
        elements.append({"x": x, "y": y, "width": w, "height": h})

    return elements
```

### 问题 3.4：坐标反算（像素坐标 → 数据坐标）

```python
def pixel_to_data(panel: dict, pixel_x: float, pixel_y: float) -> tuple:
    """
    将图表中的像素位置映射回数据坐标。
    需要面板边界 + 轴标签位置 + 刻度信息。
    """
    x_scale = (panel["x_max"] - panel["x_min"]) / panel["pixel_width"]
    y_scale = (panel["y_max"] - panel["y_min"]) / panel["pixel_height"]

    data_x = panel["x_min"] + pixel_x * x_scale
    data_y = panel["y_min"] + pixel_y * y_scale

    return (round(data_x, 1), round(data_y, 1))
```

## 关键设计决策

| 决策 | 选型 | 原因 |
|------|------|------|
| OCR 引擎 | Tesseract + pytesseract | 开源、离线、无多模态依赖 |
| PDF 渲染 | pypdfium2 | 比 PyMuPDF 更开放的开源协议 |
| 图表分析 | OpenCV (cv2) | 颜色分割、轮廓检测、直线检测 |
| 布局分析 | 纯规则 + 投影分析 | 无需训练数据，可控可解释 |
| 输出格式 | Markdown + JSON | Markdown 给人读，JSON 给 AI 读 |
